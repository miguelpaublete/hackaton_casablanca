# Crear hilo para compartir layouts de data X con seguridad

## Objetivo
Facilitar al equipo de seguridad el acceso a los layouts de todos los data X que forman parte de la solución para que puedan avanzar en su dictamen.

## Implementation Notes
Se debe crear un hilo de comunicación donde se compartan los layouts definidos y se mantenga informado al equipo de seguridad.

## Acceptance Criteria
- [ ] Hilo de comunicación creado y compartido con el equipo de seguridad.
- [ ] Todos los layouts de data X están disponibles en el hilo.
- [ ] El equipo de seguridad confirma recepción y acceso a la información.

## Test Plan
Comprobar la existencia del hilo, la disponibilidad de los layouts y la confirmación del equipo de seguridad.
