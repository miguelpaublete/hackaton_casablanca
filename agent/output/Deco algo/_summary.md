# Resumen de la reunion

En la reunión semanal del proyecto Deco Algo se revisó la liberación del modelo y el Acuerdo de Nivel de Servicio (MSA), confirmando el inicio de los desarrollos definidos por IMETRIX. Se discutió el circuito de aprobación del MSA, ajustes en el calendario de reuniones, y se solicitó al equipo de seguridad compartir los layouts de todos los data X para avanzar en su dictamen. Se acordaron tareas como compartir el MSA, consultar el circuito de aprobación, convocar una reunión en México y crear un hilo para layouts de seguridad.
