# Ejecutar casos de uso con ventas en Max Trader

## Objective
Aprovechar el tiempo disponible para ejecutar casos de uso en Max Trader que solo requieren ventas.

## Implementation Notes
- Priorizar los casos de uso que no dependan de la funcionalidad de compras.
- Coordinar con el equipo de pruebas para optimizar el tiempo disponible.

## Acceptance Criteria
- [ ] Ejecución exitosa de los casos de uso con ventas.
- [ ] Documentación de los resultados obtenidos durante las pruebas.

## Test Plan
- Validar que los casos de uso ejecutados cumplen con los criterios de éxito definidos.
- Revisar los resultados para identificar posibles incidencias adicionales.
